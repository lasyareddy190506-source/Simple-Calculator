# Simple-Calculator
"A beginner-friendly Java console-based calculator that performs basic arithmetic operations: addition, subtraction, multiplication and division. Built to practice Java programming, user input handling and conditional logic."

## Features
- Addition, Subtraction, Multiplication, Division
- Handles division by zero
- Console-based input/output

## Technologies Used
- Java
- Console I/O
